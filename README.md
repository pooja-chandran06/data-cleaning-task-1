# Task 1: Data Cleaning and Preprocessing

## 📌 Dataset
Netflix Movies and TV Shows dataset (Kaggle)

## 🎯 Objective
To clean and preprocess raw data by handling missing values, duplicates, and inconsistent formats.

## 🛠 Tools Used
- Python
- Pandas

## 🔍 Steps Performed

1. Identified missing values using `.isnull()`
2. Filled missing values with appropriate placeholders
3. Removed duplicate records
4. Standardized text values (converted to lowercase)
5. Converted date format to datetime
6. Renamed column headers (lowercase, underscores)
7. Checked and fixed data types

## 📊 Output
Cleaned dataset saved as:
`cleaned_netflix_data.csv`

## 📚 Learning Outcome
- Learned how to clean real-world datasets
- Understood handling missing values and duplicates
- Improved data preprocessing skills using Pandas
